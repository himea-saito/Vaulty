## Vaulty Addon ReadMe ##
Author: Himea
Version: 1.3-release
Updated: 1/10/2023
WoW Version: 10.0.2

Vaulty will load the Great Vault UI that is normally hidden and requires you to walk to Oribos to see.

To open the frame, you can click the minimap button or type "/vaulty", "/gv", or "/greatvault".

Download from:
https://www.curseforge.com/wow/addons/vaulty
https://github.com/himea-saito/Vaulty

Use the GitHub URL to submit issues.
